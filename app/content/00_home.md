## Proyecto
Este dashboard convierte el análisis del notebook en una experiencia web navegable, clara y profesional.

### Identidad visual
La interfaz se ha diseñado con una paleta inspirada en la Antártida:
- blanco nieve como base
- azul profundo para títulos y jerarquía
- azul oceánico para interacción
- azul cielo y azul hielo para contraste visual
